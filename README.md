# flight-data-analysis
Analysing flight dataset using AWS by creating automated end-to-end data cleaning pipeline, running SQL queries, and creating interactive dashboards to draw and visualize insights.
AWS Glue and Amazon S3 is used to craete the end-to-end automated data cleaning pipeline.
AWS Athena is used to run SQL queries on the data to draw insights.
AWS Quicksight is used to create interactive dashboards with plots and graphs to visualize insights and better understanding.
A report is made describing the complete steps and procedure of this whole project along with discussing the insights and crucial informations with images and visual infographics.
